# MrGreenAuto Site

Эта папка подготовлена как структура для GitHub Pages.

- `index.html` — главный файл сайта (сейчас тут заглушка, замените на полный код из чата).
- `img/` — положите сюда картинки автомобилей и запчастей.
- `data/` — можно хранить JSON с лотами.

## Деплой на GitHub Pages
1. Создайте репозиторий и загрузите все файлы.
2. Включите GitHub Pages (Settings → Pages → Deploy from branch).
3. Ваш сайт станет доступен по адресу https://USERNAME.github.io/REPO/
